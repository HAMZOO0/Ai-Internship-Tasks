# AI Internship Tasks

This repository contains the documentation for three AI internship tasks.

## Task 1: Iris Dataset Exploration

This script performs an exploratory data analysis (EDA) on the classic Iris dataset. It loads the data, provides summary statistics, and visualizes the relationships between different features.

## Task 2: Stock Price Prediction

This script predicts the next day's closing price of Tesla (TSLA) stock using a simple linear regression model. It fetches historical data from Yahoo Finance, prepares the data, trains the model, and evaluates its performance.
